# Privacy Policy for TG-Help Support Bot

**Last Updated:** February 2024

## Data Collection
This bot collects and stores only the necessary data to provide support services:
- Telegram User ID
- Username or First Name
- Support ticket messages
- Ticket creation time and status

## Data Usage
The collected data is used solely for:
- Managing support tickets
- Facilitating communication between users and admin
- Tracking ticket status and history
- Generating anonymous statistics

## Data Storage
- Data is stored in a JSON file on Heroku servers
- No personal data is shared with third parties
- Users can request deletion of their data

## Data Retention
- Ticket data is retained for operational purposes
- Users can request data deletion via Telegram

## Contact
For privacy concerns, contact via Telegram bot.
